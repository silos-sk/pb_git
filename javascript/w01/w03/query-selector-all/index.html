Query Selector All Exercise

Query Selector all always returns a node list. A node list is similar but has some differences to an array

Here are the similarities between arrays and node lists.

  • Node lists have a length property
  • Node lists are zero indexed
  • You can access individual nodes using square brackets
  • You can use forEach on nodelists

Note: You can only use forEach, other array methods like filter, map and reduce wont work on node lists your will have to convert the node list to an array first
